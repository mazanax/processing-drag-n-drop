## How to install

1. Склонируйте репозиторий (Clone this repository)
2. Откройте файл DragAndDropSketch в редакторе Processing (Open DragAndDropSketch in Processing IDE)
3. Запустите программу (Run Sketch)
4. Proffit!!!!!
